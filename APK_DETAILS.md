## 📥 How to Install

### Method 1: Android Device (Easiest)

1. **Download the APK** from the `releases/` folder
2. **Transfer to your Android phone** (email, USB, etc.)
3. **Open file manager** on your phone
4. **Locate the APK file** and tap it
5. **Tap "Install"** when prompted
6. **Done!** The app is now on your phone

> **Note:** You may need to enable "Install from Unknown Sources" in Settings → Security

### Method 2: Using ADB (Android Debug Bridge)

If you have Android SDK installed:

```bash
adb install SmartTasks-v1.0-debug.apk
```

### Method 3: Android Studio Emulator

1. Open Android Studio
2. Launch an emulator (API 24+)
3. Drag & drop the APK into the emulator window
4. Wait for installation

## 🐛 Troubleshooting

### Installation fails with "Parse error"
- Ensure you have Android 7.0 or higher
- Try downloading the APK again
- Check your device storage (need at least 50 MB free)

### App won't open after installation
- Restart your phone
- Check if you have storage permission
- Reinstall the APK

### "App not installed" error
- You may have a lower version installed. Uninstall it first
- Try Method 2 (ADB install) instead

### Reminders not working
- Check notification settings for Smart Tasks
- Ensure alarm permission is granted
- Restart the app
