# 📥 Installation Guide - Smart Tasks APK

Quick start guide for installing and using Smart Tasks.

---

## ⚡ Quick Install (2 Minutes)

### For Windows/Mac Users

1. Download `SmartTasks-v1.0-debug.apk` from the `releases/` folder
2. Connect your Android phone via USB
3. Enable USB Debugging on your phone (Settings → Developer Options)
4. Use ADB command:
   ```bash
   adb install SmartTasks-v1.0-debug.apk
   ```

### For Android Phone Users (No Computer)

1. Download the APK file to your phone
2. Open your file manager
3. Find and tap the APK file
4. Tap "Install"
5. Wait for installation to complete
6. Tap "Open" or find the app in your app drawer

### For Android Studio Users

1. Open Android Studio
2. Launch an Android emulator (must be API 24+)
3. Drag the APK file over the emulator window
4. Wait for auto-installation

---

## ✅ After Installation

### First Launch
- App may ask for permissions
- Grant necessary permissions for reminders & notifications
- Start creating your first task!

### Grant Permissions
- **Alarm & Reminders** - Enable to get task notifications
- **Storage** - May be needed for app data
- **Notifications** - Enable to receive alerts

## 🆘 Installation Troubleshooting

| Problem | Solution |
|---------|----------|
| "Unknown app developer" | Go to Settings → Security → Enable Unknown Sources |
| "Parse error" | Device must run Android 7.0+ |
| "File not found" | Check file download location |
| "Insufficient storage" | Free up at least 50 MB |
| "App won't install" | Try uninstalling old version first |
